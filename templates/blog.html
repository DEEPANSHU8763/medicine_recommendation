In the age of tech and remote healthcare, we developed a symptom-based diagnosis app using Flask and ML.

The Problem

People need quick, reliable answers when symptoms appear. Overlapping signs make self-diagnosis hard.

The Solution

  1. Symptom Input: Simple interface for inputting symptoms.
  2. ML Model: Trained Decision Tree predicts diseases.
  3. Instant Prediction: Users get immediate diagnosis suggestions.
  4. Extra Info: Each prediction includes tips, precautions, diet, and workouts.

Key Features

  • User-Friendly: Clean interface.
  • Accurate: Uses real training data.
  • Comprehensive: Info-rich support for user guidance.
  • Educational: Health-related content included.

Conclusion

This app bridges tech and healthcare. It empowers people with tools to better understand their health and encourages awareness.
